# 第6回課題: YouTube集中モード v2 (プロトタイプv2)

「YouTubeを見始めると止まらず、勉強や課題の時間がなくなる」という課題を解決するためのプロダクト **「YouTube集中モード」** が、第6回課題用として **v2** へ進化しました！

v2では、単なるタイマーだけでなく、実際に視聴した後に **「作業に戻れたか？」という振り返り (フィードバック)** を行い、その結果を自己分析できるようにログ機能を追加しました。

---

## 🚀 v1 から v2 への主な変更点・追加機能

| 機能 | v1 (初期プロトタイプ) | v2 (プロトタイプv2) |
|---|---|---|
| **タイマー機能** | 理由と視聴時間を入力してカウントダウン | そのまま維持 (一時停止・リセットも可能) |
| **アラーム警告** | 終了時にビープ音で警告 | そのまま維持 (Web Audio API による動作) |
| **振り返り画面** | 「作業に戻りました」ボタンのみ | **「戻れた (成功)」と「戻れなかった (失敗)」の2択ボタンに変更** |
| **データ保存** | 保存なし (セッション限り) | **`localStorage` を利用し、利用履歴をブラウザに永続保存** |
| **統計表示** | なし | **本日の利用回数、本日の成功回数、本日の成功率を表示** |
| **履歴ログ一覧** | なし | **直近10件の利用ログを一覧表示（スクロール対応）** |
| **ログ削除機能** | なし | **特定のログの個別削除 (`×` ボタン) & 全ログの一括クリアに対応** |
| **検証サポート** | 1分以上のタイマーのみ | **`0.1` 分 (6秒) などの小数値を入力可能にし、テスト検証を容易に** |
| **デザイン** | `480px` 幅のシンプルダークUI | **`540px` 幅に拡張、統計・履歴カードを美しく統合したシンプルモダンUI** |

## 📱 アプリの使い方

1. **ルールの設定 (初期画面)**
   * 「今YouTubeを見る理由」を入力します（例: `英語のリスニング解説動画を1本見る`）。
   * 「視聴時間 (分)」を入力します。
     * 💡 **ヒント**: `0.1` と入力すると、**6秒間** のテスト用タイマーとして動作を確認できます。
   * 「集中モードを開始する」ボタンを押すと、タイマー画面に切り替わります。

2. **視聴とカウントダウン (タイマー画面)**
   * 設定された理由が上部に常時表示され、自己抑制を促します。
   * 残り時間が大きくカウントダウン表示されます。
   * 残り1分を切ると、タイマーが赤く点滅して警告します。
   * 途中で諦める場合は「あきらめてリセット」を押すとログを記録せずに戻れます。

3. **振り返りと記録 (タイマー終了画面)**
   * 時間が終了するとアラーム（ピピッ、ピピッ）が繰り返し鳴ります。
   * 画面が終了画面に変わり、**「すぐに作業に戻れましたか？」** と質問されます。
   * YouTubeを閉じてすぐに作業に戻れたら **「👍 戻れた！ (成功)」**、ダラダラ見続けてしまったら **「👎 戻れなかった (失敗)」** をクリックします。
   * クリックした時点で、結果が自動的にブラウザの `localStorage` に記録され、アラームが止まって初期画面に戻ります。

4. **統計と履歴の確認 (初期画面下部)**
   * 初期画面に戻ると、「本日の統計」および「直近の利用ログ」が即座にアップデートされます。
   * **「本日の統計」**: 今日1日の中で、何回YouTubeを見て、何回作業に戻れたか、成功率（パーセンテージ）が表示されます。
   * **「直近の利用ログ」**: 過去最大10件までの利用記録（時間、理由、結果）が表示されます。不要なログは `×` ボタンで個別に消すことができます。
   * 全てのデータをリセットしたい場合は「ログをクリア」を押します。

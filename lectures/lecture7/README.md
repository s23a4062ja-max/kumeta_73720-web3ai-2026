# 第7回課題: YouTube集中モード v3 (最終発表会用プロトタイプ)

「YouTubeを見始めると止まらず、勉強や課題の時間がなくなる」という社会（学生）バグを解決するためのプロダクト **「YouTube集中モード」** が、第7回課題用として **v3** へと進化しました！

v3では、自律性をさらに高めるための **「AIアドバイザーによる振り返り機能」** や **「集中タイプ診断」** を追加し、さらにアプリ単体で画面共有をしてデモ発表ができる **「最終発表まとめ」プレゼンタブ** を統合しました。

---

## 📈 v1, v2, v3 の進化と機能差分

本プロジェクトは、VPC（バリュープロポジションキャンバス）の設計から始まり、段階的に機能を拡張してきました。

| バージョン | 主なテーマ | 主な追加機能 | 得られるユーザー体験・効果 |
|---|---|---|---|
| **v1** | **目的の宣言と制限** | ・見る「理由」と「時間」の事前宣言<br>・大きく表示されるタイマー<br>・終了時のアラーム音 | 視聴の目的を自覚させ、ダラダラ視聴に入ってしまう前に強制的にブレーキをかける。 |
| **v2** | **振り返りと蓄積** | ・「作業に戻れた/戻れなかった」振り返り確認<br>・`localStorage` を使ったログの蓄積<br>・「本日の統計（回数、成功率）」と履歴表示 | 自分の日々の行動をデータとして可視化し、自分がどのくらい誘惑に勝てているのかを客観視させる。 |
| **v3** | **AI分析と発表統合** | ・理由や結果に応じた **「AI風アドバイス」**<br>・本日の実績から診断する **「集中タイプ診断」**<br>・履歴ログそれぞれに対する **「個別AI一言アドバイス」**<br>・アプリ内 **「最終発表まとめ（スライド機能）」** | データから一歩踏み込んで、行動を振り返るためのアドバイスを提供。さらにアプリ自身が発表スライドになる。 |

---

## 🤖 v3 の主な新機能

1. **AI風振り返りアドバイス ( setupScreen 上部 )**
   * タイマー終了時の振り返りで結果（成功/失敗）を選択すると、**「視聴した理由」「設定時間」「今日のこれまでの成功率」** をJavaScriptで多角的に分析し、次回の行動を改善するためのアドバイスをリアルタイムに生成して表示します（外部API非依存のため即座に動作します）。
2. **集中タイプ診断 ( 統計エリア )**
   * 今日のこれまでの利用ログから、現在の集中度合いを4つのタイプに分類して診断します。
     * 🚀 `超集中キープ型` : 3回以上の利用で成功率が80%以上。
     * ✨ `安定スタート型` : 利用回数は少ないが、全て成功している状態。
     * 🔄 `リカバリー挑戦型` : 成功率50%〜79%の間で奮闘している状態。
     * ⚠️ `脱線注意型` : 成功率が50%未満の誘惑危険状態。
3. **履歴ごとの一言AIアドバイス**
   * 履歴ログ一覧の各ログカードの中に、その時の理由と成功結果に合わせた1行の「AIメモ」が自動添付されます。
4. **「最終発表まとめ」プレゼンタブ**
   * アプリ上部に「アプリ本体」と「最終発表まとめ」の切り替えタブを追加。
   * 「解決したいバグ」「対象ユーザー」「進化のプロセス」「今後のロードマップ」がスライド風にカードレイアウトされており、発表会でこの画面を見せるだけでデモ発表が可能です。

## 🎤 最終発表会で伝えるべき発表の構成案

発表会でデモを行う際は、以下の流れで説明すると説得力が増します。

1. **自己紹介と解決したい課題 (バグ)**
   * 「大学生の多くが『勉強しようとしたのにYouTubeをダラダラ見て時間を無駄にした』というバグを抱えています。これを解決するために開発しました。」
2. **VPCに基づくアプローチ (バリュー)**
   * 「ただタイマーで止めるだけでなく、ユーザーに『なぜ見るのか』を言語化させ、さらに終わった後に『本当に戻れたか』を自己申告させることで、認知行動療法的に自制心を鍛えます。」
3. **デモストレーション**
   * タブを「アプリ本体」に切り替えます。
   * 視聴時間を `0.1` 分（6秒）に設定し、理由に「勉強動画をみる」と入力してスタートします。
   * カウントダウン終了後、アラームが鳴り、結果選択画面へ。「戻れた！」を選択します。
   * 初期画面に戻ると、**「AIアドバイザーのアドバイス」** が出現し、**「本日の統計」** と **「集中タイプ」** が更新される様子をアピールします。
4. **将来のビジョン (ロードマップ)**
   * タブを「最終発表まとめ」に切り替えます。
   * 「今後はGemini等の本物のLLMを組み込み、ダラダラ見てしまった言い訳に対してAIがパーソナルコーチとして寄り添う機能や、Chrome拡張機能と連携して実際にタブを閉じる仕組みを実装したいです」と伝えます。
